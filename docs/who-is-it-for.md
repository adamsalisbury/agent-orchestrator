# Who Is It For?

## Target Users

Agent Orchestrator is designed for anyone who wants to explore multi-agent AI workflows — particularly developers, technical leads, and teams experimenting with AI-assisted software development.

### Developers and Engineers

- Set up a team of specialised AI agents to assist with different aspects of a project
- Have a "backend engineer" agent handle API design while a "frontend developer" agent works on UI
- Agents can consult each other, mirroring how real development teams collaborate

### Technical Leads and Architects

- Define a project context and let agents understand the broader goals
- Use agents with different specialisations to get varied perspectives on design decisions
- Track all agent interactions and consultations in threaded conversations

### AI and Automation Enthusiasts

- Experiment with multi-agent orchestration patterns
- Observe how delegation chains work when agents route tasks to teammates
- Explore prompt engineering through custom agent personas

### Solo Developers

- Create a virtual team of specialists to fill knowledge gaps
- Get feedback from agents with different expertise areas without needing human teammates
- Use the project context feature to keep all agents aligned on goals

## Use Cases

- **Software development** — create agents for frontend, backend, DevOps, QA, and let them collaborate on a codebase
- **Code review** — set up a reviewer agent with specific quality standards and have it review work
- **Research and analysis** — create agents with different domain expertise to analyse a problem from multiple angles
- **Learning and exploration** — interact with specialised agents to learn about unfamiliar topics in context
- **Prototyping** — rapidly iterate on ideas with a team of agents that can read and write to a shared workspace
